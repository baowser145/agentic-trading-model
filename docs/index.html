Data source

Default on GitHub Pages: bundled sample_snapshot.json (demo only). Live needs a reachable HTTPS status URL (GitHub Pages is HTTPS — browsers block http://localhost mixed content). Use a tunnel (ngrok / Cloudflare) or open the local watch UI instead.

How to wire live data 
# On your machine (paper bot)
python -m agentic_trading run-loop --quotes live --session-dir logs/paper_live --interval 60

# Local UI (easiest — no GitHub needed for live)
python -m agentic_trading watch --session-dir logs/paper_live
# → http://127.0.0.1:8787/
